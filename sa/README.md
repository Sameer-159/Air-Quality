# Air-Quality# Air Quality Analysis Dashboard

A comprehensive web application for analyzing air quality data using Python, Prolog, and modern web technologies.

## Features

- **Interactive Data Visualization**: Time series charts, distribution analysis, and correlation plots
- **AI Reasoning with Prolog**: Rule-based pollution event detection
- **Statistical Analysis**: Mean, median, standard deviation, and anomaly detection
- **Real-time Updates**: Dynamic filtering and analysis
- **Responsive Design**: Works on desktop and mobile devices

## Tech Stack

- **Backend**: Python Flask
- **AI Reasoning**: Prolog (SWI-Prolog)
- **Frontend**: HTML5, CSS3, JavaScript (Chart.js)
- **Data Processing**: Pandas, NumPy
